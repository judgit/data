# 行政事業レビューシートLOD

内閣官房によって公開されている「[行政事業レビューシートの主要事項のデータベース](https://www.gyoukaku.go.jp/review/review.html)」をLOD化したものです。

本データは[JUDGIT!](https://judgit.net/)で使用されています。
